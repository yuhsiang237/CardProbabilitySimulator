# CardProbabilitySimulator
抽卡模擬器，簡易模擬SSR、R、N三張卡片的抽取。  
設定卡片機率後抽取1000000次。

採動態記憶體配置，若需額外添加/減少卡片僅需修改卡片陣列。

編譯器環境C99。
